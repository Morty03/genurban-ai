# GenUrban AI

Generative Satellite Vision system for simulating future urban morphology using satellite imagery and machine learning.

## Overview
GenUrban is an AI-driven system that analyzes satellite imagery and urban features to simulate possible future urban growth patterns.

## Features
- Satellite imagery preprocessing
- Feature extraction (urban density, vegetation, edges)
- Machine learning modelling for urban prediction
- Interactive frontend visualization

## Tech Stack
Python, FastAPI, React, GIS, Machine Learning

## Project Structure
backend/ – API and server logic  
genurban-frontend/ – React dashboard  
ml_engine/ – ML models and preprocessing pipelines
